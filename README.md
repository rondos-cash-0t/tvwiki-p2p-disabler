# Webtorrent-p2p-disabler

## Webtorrent p2p disabler

뉴토끼 (toki25, sbxh9 등) 는 애니메이션 플레이어 부분의 P2P 만 비활성화 됩니다.

최근 추가되었다가 사라진 이미지 P2P 는 별도 작업이 필요하며, 해당 부분은 핵심적인 코드를 수정해야 하므로, 되도록이면 유저스크립트를 작성하지 않을 계획입니다.
