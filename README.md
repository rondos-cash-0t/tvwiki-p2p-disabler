# Webtorrent-p2p-disabler

## Userscript

[Add Userscript](https://github.com/rondos-cash-0t/webtorrent-p2p-disabler/raw/refs/heads/main/p2p-disable.user.js)

## 설명

이 유저스크립트는 영상 스트리밍 사이트 일부에서 사용하는 [Swarmcloud](https://swarmcloud.net/) 등의 Webtorrent/P2P 기능을 비활성화하는 유저스크립트입니다.

현재는 일부 사이트만 match 에 추가하였습니다.

## 참고사항

뉴토끼 (toki25, sbxh9 등) 는 애니메이션 플레이어 부분의 P2P 만 비활성화 됩니다.

최근 추가되었다가 사라진 이미지 P2P 는 별도 작업이 필요하며, 해당 부분은 핵심적인 코드(XXXXX-images 의 데이터나 특정 script 비활성화)를 수정해야 하므로, 되도록이면 유저스크립트를 작성하지 않을 계획입니다.
